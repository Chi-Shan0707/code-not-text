## 📊 **研究发现的重要性**

### **主要发现**

#### **1. 数学领域：强烈信号** 🎯
```
AoA: 0.958 [0.931, 0.980]
Reranking: +10.0% pass@1
机制：特征捕捉"推理不收敛"模式
```
**价值**：证实了process supervision在数学推理中的有效性

#### **2. 科学领域：窄信号** ⚠️
```
AoA: 0.799 [0.775, 0.822]
Reranking: +8.0% pass@1
机制：主要由token confidence驱动，非轨迹结构
```
**价值**：揭示了不同domain的信号来源差异

#### **3. 编码领域：信号失效** ❌
```
AoA: 0.434 [0.404, 0.464]
Reranking: -0.6% (no improvement)
机制：执行依赖的正确性无法从文本轨迹获取
```
**价值**：重要负结果，指出了方法论的局限性
